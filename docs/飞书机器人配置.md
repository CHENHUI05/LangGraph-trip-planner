## 3.应用配置

### 1.权限管理

点击开通4个权限

```
获取与发送单聊、群组消息
获取与发送单聊、群组消息
读取用户发给机器人的单聊消息
以应用的身份发消息
```

![5](..\image\飞书5.jpg)

### 2.事件与回调

这里的地址需要我们先申请内网穿刺（[ZeroNews文档](ZeroNews内网穿刺.md)）最后得到的映射地址，注意需要的是**后端的映射地址**）

```python
https://xxxxxxxxx.fy.takin.cc
```

将下面的地址，也就是**飞书的后端接口**填入下方，点击保存（**注意：点击保存之前确保我们已经启动了项目的后端已经我们配置的ZreoNews映射**）

```python
https://xxxxxxxxx.fy.takin.cc/api/feishu/webhook
```

![6](..\image\飞书6.jpg)

然后点击添加事件，搜索**接收消息**，点击添加后我们的飞书机器人就配置成功了
